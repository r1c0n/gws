CORS Test for Gamma Web Server

This page tests CORS by making requests to your GWS server from a different origin (port).

Instructions:

  1. Make sure GWS is running on port 80 (localhost)
  2. Open this file using a different port or server (e.g., python -m http.server 8000)
  3. Enable CORS in config.json by setting "cors": { "enabled": true }
